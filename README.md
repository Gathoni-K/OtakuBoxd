##  Features

- Track currently watching, completed, and planned anime
- Save and update anime progress
- Mark anime as completed
- Write and view anime reviews
- Clean, responsive, anime-focused UI

---

##  Purpose

OtakuBoxd was built to:
- Practice building **feature-rich React applications**
- Work with third-party APIs and structured media data
- Implement user-centric state management
- Explore social and content-driven UI patterns

The project focuses on scalability, clarity, and user experience.
